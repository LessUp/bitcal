High-Performance SIMD Library

SIMD-Accelerated Bit Manipulation

Modern C++17 header-only library with automatic SSE2/AVX2/NEON acceleration. Up to 6× faster than scalar implementations with zero runtime overhead.

Get Started View on GitHub
example.cpp 
#include <bitcal/bitcal.hpp>

int main() {
    // Create 256-bit bit arrays
    bitcal::bit256 a(0xDEADBEEF);
    bitcal::bit256 b(0xCAFEBABE);

    // SIMD-accelerated operations (~2.1ns)
    auto c = a & b;           // AND
    auto d = a | b;           // OR
    auto e = a ^ b;           // XOR

    // Hardware-accelerated bit counting
    auto pop = c.popcount();              // Count set bits
    auto lz = c.count_leading_zeros();    // Leading zeros

    return 0;
}

Why BitCal?

Designed for performance, built for developers

SIMD Acceleration

Automatic selection of SSE2, AVX2 (x86), or NEON (ARM) instruction sets. No code changes needed.

Zero Overhead

Compile-time dispatch via if constexpr. All optimization happens at compile time, zero runtime cost.

Header-Only

Single #include, no compilation, no dependencies. Copy headers and start coding immediately.

Rich API

Bitwise operations, shifts, popcount, CLZ/CTZ, bit reversal, ANDNOT. Everything you need.

Cross-Platform

Linux, Windows, macOS on x86-64, ARM64, and ARM32. CI verified on all platforms.

Type Safe

Compile-time bit-width validation with constexpr. Catches errors at compile time.
