# 更新日志

## v2.1.0 (2026-02-27)

### 新增功能

- **ANDNOT 操作** — `bitarray::andnot(mask)` 使用原生 SIMD 指令
- **bit1024 测试** — 完整的 1024 位操作测试覆盖
- **性能测试套件** — benchmarks 目录

### 性能优化

| 优化项 | 说明 |
|--------|------|
| `operator~` | 128/256/512-bit 使用 SIMD NOT 指令 |
| `is_zero()` | 使用 `_mm_movemask_epi8` / `_mm256_testz` 单指令检测 |
| `clear()` | 改用 `std::memset` |
| `reverse()` | 原地操作，消除临时数组 |

### Bug 修复

| 问题 | 修复 |
|------|------|
| MSVC SSE2 宏检测 | 添加 `_M_X64` 判断 + `<intrin.h>` |
| NEON `vmvnq_u64` | 改用 `veorq_u64` XOR 全 1 |
| NEON 运行时移位 | 改用 `vshlq_u64` 支持变量 |
| SSE 256-bit shift | 修复 4-qword 进位链断裂 |
| AVX 256/512-bit shift | 修复 cross-lane carry 传播 |

### CI/CD

- 添加 ARM32 交叉编译测试
- CMake 自动检测 SIMD 编译标志

## v2.0.0 (2026-01-08)

### 完全重构

从 OOP 继承设计重写为现代 C++17 模板设计。

### 架构对比

**旧设计**:
```cpp
BitCal::getInstance().bitAnd64(a, b);  // 运行时分发
```

**新设计**:
```cpp
bitcal::bit256 result = a & b;  // 编译期优化
```

### 新增功能

| 功能 | 说明 |
|------|------|
| Header-only | 零编译依赖 |
| 编译期 SIMD | 自动选择最优后端 |
| ARM NEON | 完整的 128/256/512-bit 支持 |
| 运算符重载 | `&`, `\|`, `^`, `~`, `<<`, `>>` |
| 位计数 | `popcount`, `clz`, `ctz` |
| 位操作 | `get/set/flip_bit`, `is_zero`, `reverse` |

### 性能数据

| 平台 | 操作 | 加速比 |
|------|------|--------|
| x86 AVX2 | AND-256 | **5.9×** |
| x86 AVX2 | XOR-512 | **5.8×** |
| ARM NEON | AND-128 | **2.6×** |

### 移除

- 虚函数开销
- 单例模式
- 外部依赖 (spdlog)
