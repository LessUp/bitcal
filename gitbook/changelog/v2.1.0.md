# v2.1.0 (2026-02-27)

## 正确性修复

- **MSVC SSE2 宏检测** — `config.hpp` 添加 `_M_X64` 判断 + `<intrin.h>`
- **NEON `vmvnq_u64`** — 标准 NEON 无此 intrinsic，改用 `veorq_u64` XOR 全 1
- **NEON `vshlq_n_u64`** — 改用 `vshlq_u64` + `vdupq_n_s64` 支持运行时变量
- **SSE 256-bit shift** — 修复 4-qword 进位链断裂（carry 传播 bug）
- **AVX 256/512-bit shift** — 修复 `_mm256_slli_si256` per-lane 操作导致的 cross-lane 错误

## 性能优化

- 新增 `load_aligned` / `store_aligned`（SSE/AVX）
- `operator~` 对 128/256/512-bit 使用 SIMD NOT 指令
- `is_zero()` 使用 `_mm_movemask_epi8` / `_mm256_testz` 单指令检测
- `clear()` 改用 `std::memset`
- `reverse()` 原地化（消除临时数组）

## 新增功能

- **ANDNOT 操作** — `bitarray::andnot(mask)` 利用原生 `vpandn` / `vbicq` 指令
- **统一二元操作分派** — `dispatch_binop<Op>` 模板

## 工程改进

- CMake 自动检测 SIMD 编译标志
- C++ 标准统一为 C++17
- 测试框架重构 + 30+ 测试用例
- Benchmark 添加 warmup + DoNotOptimize
- 文档更新
