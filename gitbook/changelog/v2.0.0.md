# v2.0.0 (2026-01-08)

## 完全重构

从 OOP 继承 + 虚函数设计重写为现代 C++17 模板设计。

## 主要变更

- **架构重写** — 从 `BitCalBase/BitCalGpr/BitCalXmm` 继承体系改为 `bitarray<Bits, Backend>` 模板
- **Header-only** — 零编译依赖，直接 `#include` 即可使用
- **编译期 SIMD 选择** — 消除虚函数调用和单例访问的运行时开销
- **完整 ARM 支持** — 128/256/512-bit NEON 实现
- **新增 API** — `get_bit`/`set_bit`/`flip_bit`/`is_zero`/`clear`/`reverse`/`byte_swap`
- **运算符重载** — `&`、`|`、`^`、`~`、`<<`、`>>`
- **位计数** — `popcount`、`count_leading_zeros`、`count_trailing_zeros`
